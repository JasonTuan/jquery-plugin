Typescript Jquery Plugin Template
===

# NPM Install
```shell
npm init
```

# Dependencies
```
npm i jquery --save
```

# DEV Dependencies
```
npm i typescript @types/jquery uglify-js rollup rollup-plugin-babel rollup-plugin-sourcemaps rimraf cpx --save-dev
```

# Greeting Plugin
```shell
npm i -d
npm run build
```
